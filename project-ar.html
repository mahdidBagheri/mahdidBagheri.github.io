AR Neurosurgery Navigation

HoloLens 2 Unity3D Deep Learning

A novel solution enabling surgeons to use HoloLens 2 to view various modalities augmented on real anatomy as holograms. The project investigated Deep Learning approaches (RPM-Net, GeoTransformer) to solve the critical challenge of point cloud registration.

Automatic Point Cloud Registration Process

Key Contributions

  • Proposed automatic point cloud registration using robust ICP modification.
  • Developed calibration methods for inside-out (HoloLens) and outside-in (HTC Vive) tracking.
  • Designed assessment methods for measuring registration/tracking errors.
