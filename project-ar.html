AR Neurosurgery Navigation

HoloLens 2 Unity3D Deep Learning

A novel solution enabling surgeons to use HoloLens 2 to view various modalities augmented on real anatomy as holograms. Investigated Deep Learning approaches (RPM-Net, GeoTransformer) for point cloud registration.

Live demonstration of Point Cloud Registration

Key Contributions

  • Proposed automatic point cloud registration using robust ICP modification.
  • Developed calibration methods for inside-out (HoloLens) and outside-in (HTC Vive) tracking.
  • Designed assessment methods for measuring registration/tracking errors.
